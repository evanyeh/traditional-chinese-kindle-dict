Installation

  1. Connect Kindle to computer via USB cable.
  2. Copy moe_dict_concised.mobi to (Kindle):\documents\dictionaries
  3. Disconnect USB cable.
  4. On the Kindle, select
    1. Settings
    2. All Settings
    3. Language & Dictionaries
    4. Dictionaries
    5. Chinese
    6. Traditional Chinese Dictionary (Taiwan Ministry of Education Concised)

Usage

While reading, long press and highlight a character or phrase. A definition box will appear with pronunciation and definition.

To view the dictionary itself, it should appear in the library.
